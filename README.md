# Intelligent Data — Zapier Integration

Zapier integration for the [Intelligent Data API](https://portal.smartvmapi.com) by apexanalytix.

## Actions

- **Address Validation** — Validate and standardize postal addresses
- **Tax ID Validation** — Validate tax identification numbers (100+ countries)
- **Bank Account Validation** — Verify bank account details
- **Business Registration Lookup** — Look up company registration data
- **Sanctions Screening** — Screen against OFAC, EU, UN sanctions lists
- **Disqualified Directors Check** — Check for disqualified directors

## Authentication

Uses API Key authentication. Get your key at [portal.smartvmapi.com](https://portal.smartvmapi.com/auth/register).

## Development

```bash
npm install
zapier test
zapier push
```
